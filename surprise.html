🛠️ Step 1: Build the Brain

Open the Shortcuts app on your iPhone and tap the + at the top right to create a new shortcut. Search for and add these actions exactly in this order:

  1. List: Add the "List" action. Copy and paste some quotes from the bottom of this page into the list items.
  2. Get Item from List: Add this action. Tap on the word "First Item" and change it to Random Item.
  3. Show Notification: Add this action. Tap the text and select your Item from List variable. Tap the arrow next to it and change the Title to: Excuse me, future architect... 🌿
  4. Wait: Add this action. Set the timer to 120 seconds.
  5. Show Alert: Add this action. Change the Title to: 🛑 Time's Up, Ma'am! and the message to: I am revoking your scrolling privileges. Drink some water and get back to your studio work!
  6. Go to Home Screen: Add this action. (This is the kicker!).

At the very top, rename this shortcut to Study Nudge 🪴 and hit Done.

⚡️ Step 2: Set the Trigger

Now we just need to tell your phone when to use this brain!

  1. In the Shortcuts app, tap the Automation tab at the bottom.
  2. Tap the + button at the top right, and select App.
  3. Tap "Choose" and select your worst distraction apps (Instagram, TikTok, etc.).
  4. Select Is Opened and make sure you check Run Immediately.
  5. Tap Next, and select the Study Nudge 🪴 shortcut you just built.

That's it! Go open Instagram and see what happens. 😉
